# Credit Risk Analysis Project

## Overview
This project analyzes customer loan data to understand credit risk patterns and predict the likelihood of loan default using data analysis and machine learning techniques.

## Dataset Description
The dataset contains financial and customer loan information used for credit risk evaluation. It includes:

- Client_ID: Unique identifier
- Age: Borrower age
- Income: Monthly income
- Loan_Amount: Amount borrowed
- Credit_Score: Customer credit rating
- Branch: Loan issuing branch
- Loan_Officer: Officer managing the loan
- Loan_Cycle: Number of previous loans
- Outstanding_Balance: Remaining unpaid amount
- Total_Paid: Amount already repaid
- Loan_Status: Target variable (Paid / Default)

## Objective
To identify patterns in borrower behavior and build a model that predicts whether a customer will default or fully repay a loan.

## Key Focus Areas
- Relationship between income and loan repayment
- Impact of credit score on default risk
- Branch-level risk comparison
- Borrower repayment behavior trends
